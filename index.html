Programa em 8 semanas

Garantir minha vaga
Semana 1

Estrutura de projeto profissional

Módulos, pacotes, imports, virtualenv, Git.

Entregável: estrutura base do app (core/, data/, ui/, main.py).

Semana 2

Persistência de dados

JSON/CSV e introdução a SQLite; CRUD de tarefas.

Entregável: adicionar, listar, concluir e remover tarefas.

Semana 3

Automação de tarefas

Arquivos/pastas (os/pathlib), planilhas (openpyxl), e‑mail, agendamento.

Entregável: relatório semanal + envio automático por e‑mail.

Semana 4

Teoria + GUI básica (Tkinter)

Tipos de aplicação (CLI/GUI/Web), janelas, eventos, widgets e layouts.

Entregável: formulário + lista de tarefas funcionando.

Semana 5

GUI avançada + integração

Múltiplas janelas, diálogos, validação e conexão com persistência/automação.

Entregável: edição/remoção, filtros e exportação via GUI.

Semana 6

Integração com APIs

Requests + JSON; exibir dados externos na aplicação.

Entregável: módulo de API (ex.: clima/motivação) integrado à GUI.

Semana 7

Testes, logs e boas práticas

unittest/pytest, logging, docstrings e PEP8.

Entregável: testes principais + logs de execução.

Semana 8

Empacotamento e distribuição

PyInstaller (.exe/.app), README e apresentação.

Entregável: executável final + documentação.

BÔNUS (gravado)

Aplicação Web com Flask

Transforme o mesmo projeto em uma app web: rotas, templates (Jinja2), formulários e conexão com sua lógica/banco.

Entregável bônus: gerenciador de tarefas acessível no navegador (criar/listar/concluir/remover).

Projeto final: Gerenciador de Tarefas Inteligente

  • Interface gráfica amigável (Tkinter/CustomTkinter)
  • Automação de relatórios e e-mails
  • Integração com APIs externas
  • Persistência em arquivo/SQLite
  • Empacotamento em executável
  • Versão web bônus com Flask

FAQ

Preciso ter feito o curso básico?

É o ideal. Você deve conhecer variáveis, condicionais, laços, funções, listas/dicionários e manipulação simples de arquivos.

Este curso aborda análise de dados?

Não. Aqui o foco é aplicações (GUI, automação, APIs e empacotamento). Para dados, considere a trilha de Análise de Dados.

A aula de aplicação web é ao vivo?

É um bônus gravado que você assiste quando quiser e aplica no seu projeto.

Qual sistema operacional posso usar?

Windows, macOS ou Linux. Mostramos como empacotar seu app para uso local.

Como tiro dúvidas?

Você terá comunidade de suporte e momentos ao vivo para perguntas.
